## RULES OF ENGAGEMENT
1. Never fabricate. Pull only from this file + MASTER_AI_DOSSIER.md.
2. No em dashes anywhere.
3. Tone by track: PRODUCTION = warm & humble; SALES = confident & direct; TECH-DATA = sharp & technical.
4. Two opening styles available: GAP-ADDRESSING (one positive line about self-employment/business) or NO-MENTION (forward-looking only). Pick per job.
5. Two lengths: SHORT (3 short paragraphs, ~half page) or STANDARD (4 paragraphs, ~3/4 page).
6. Closing: NEUTRAL by default (no "available immediately" / no heavy local angle) unless a job clearly rewards it.
7. Mirror 2-4 keywords from the job description naturally. Never keyword-stuff.
8. Default business name on production/office letters = "Valley View Media." Use TRW/Tate names only on sales letters where it helps.
9. Keep claims defensible (the "explain it across a table" test).
10. One page max. Address a real hiring manager name if known; else "Dear Hiring Manager,".
11. DATES = match the resume exactly. If the resume uses padded DISPLAY dates (see Master Dossier section 3b), any dates or durations mentioned in the cover letter must use the SAME padded dates. Never state a tenure or year in the letter that conflicts with the resume for the same application. When in doubt, refer to experience in general terms ("over the past few years," "in my recent production roles") rather than specific dates.
